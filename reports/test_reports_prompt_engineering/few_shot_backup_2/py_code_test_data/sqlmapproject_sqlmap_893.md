Severity: High
CWE: CWE-79
Location: Multiple lines, particularly where `retVal` is constructed using string replacements and regex substitutions (e.g., lines involving `paramString.replace(...)`).

Remediation: Implement strict output encoding based on the final destination context of `retVal`.
